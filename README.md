## About
A repo to explore vector search
- generates data
- generates embeddings for a few fields
- searches by matching query vector and generated vector using cosine similarity.

## How to run
Make sure you have : 
- mongodb (8.0.4)
- python (3.12.9)
- poetry (2.1.1)

### Installing deps
`bin/install`
### Running Server
`bin/run dev`
